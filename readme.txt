== Description ==

Adauga codul de monitorizare Trafic.ro in site-ul tau. Trebuie sa introduci codul de monitorizare si sa-l afisezi dupa o metoda la alegere:
* [Recomandata] Editarea unui fisier al sablonului (de obicei footer.php) pentru a adauga functia de afisare a codului de monitorizare.
* [Usoara] Afisare directa in subsolul paginilor.
* [Widget] Adauga codul de monitoriza folosind widget-ul dedicat.

= Plugin's Official Site =

Trafic.ro Analytics ([http://www.slickdesign.ro/wordpress-plugins/plugin-wordpress-pentru-adaugare-cod-trafic-ro-in-site/](http://www.slickdesign.ro/wordpress-plugins/plugin-wordpress-pentru-adaugare-cod-trafic-ro-in-site/))

== Installation ==

* Descarca fisierul zip
* Dezarhiveaza in directorul `/wp-content/plugins/`
* Activeaza plugin-ul prin meniul Module din Wordpress
* Introdu codul de monitorizare Trafic.ro si alege o metoda de afisare
